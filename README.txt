Capstone Project Alessio Panconi

Il mio progetto sarà un sito vetrina per una psicologa il quale esporrà tutte le sue informazioni anagrafiche e professionali , darà anche la possibilità di poter contattare e ottenere in informazioni per un appuntamento ed eventualmente prenotarlo

-Frontend
React con Vite
React-Bootstrap e Bootstrap per la grafica e la responsività
React Router DOM per la gestione della navigazione tra le pagine

-Backend
Spring Boot 
Spring Data JPA per la gestione del database
Spring Validation per la validazione dei dati
